本章会使用Python进行一些图像处理操作。除了需要Python3.6+外，我们还需要安装一些扩展库。

Pillow

Pillow是Python当中非常受欢迎的图像处理库，我们可以在命令行运行如下代码安装它:

pip install Pillow

numpy

numpy常常用于多维数组的数值计算，我们在处理像素数值的时候会用到它，在命令行运行如下代码可以安装:

pip install numpy

python-qrcode

python-qrcode用于生成二维码图像，使用如下命令行指令进行安装:

pip install qrcode
