本章介绍一些人工神经网络开发的技术，下面列出本章节会使用的一些依赖库。

numpy

numpy是著名的Python数值计算库，许多的数据计算操作都依赖它实现。使用pip就能完成下载安装，命令如下:

pip install numpy

matplotlib

常用的Python绘图工具，方便进行数据可视化工作。pip的安装指令如下:

pip install matplotlib

Pillow

Pillow是Python编程语言中免费、开源的第三方图像处理库，支持对许多不同图像文件格式的打开、编辑和保存等操作。

Pillow的pip安装指令如下:

pip install Pillow

PyTorch

PyTorch是一个源自Torch库的开源机器学习框架，用于计算机视觉和自然语言处理等应用程序的开发。

PyTorch在linux下的cpu版本安装方式:

pip3 install torch torchvision --extra-index-url https://download.pytorch.org/whl/cpu

PyTorch在windows下的cpu版本安装方式:

pip3 install torch torchvision

PyTorch在mac下的cpu版本安装方式:

pip3 install torch torchvision

如果你想安装其它版本的PytTorch可以参看官方网站给出的建议:

https://pytorch.org/get-started/locally/
